# djreact-simple-blog
This is simple Blog apps using Django and React as Main stack this is pure learning simple project


# Running project

## Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `./manage.py makemigrations`
4. `./manage.py migrate`
5. `./manage.py runserver`


## Frontend 
1. `cd frontend`
2. `npm install`
3. `npm start`
